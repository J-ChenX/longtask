**最后更新：** 2026-06-24

## 文件结构

| 文件 | 用途 |
|------|------|
| [expert-roles.md](../../../expert-roles.md) | 专家角色、预审步骤、审查协议、严重性分级 |

## 配置与约定

| 项 | 约定 |
|----|------|
| 预审方式 | 优先平台原生 subagent；不可用时按 Codex 预审降级协议顺序执行 |
| 专家隔离 | 每个专家先独立审查，再汇总发现 |
| 严重性 | 文档过时与阻断同级 |
| 并发上限 | 由当前平台环境决定，不在技能内写死 |
| 降级记录 | 在 `_ACTIVE.md` 标记顺序执行，每位专家写入独立小节 |
